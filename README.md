# Power BI Smart Q&A Visuals
## Interactive Dashboard Using Natural Language Queries
##  Project Overview
This project showcases an interactive Power BI dashboard that allows users to generate insights simply by typing questions. Using Power BI’s built-in Q&A visual, the report automatically creates charts such as bar charts, line charts, and tables — without requiring DAX or manual coding.
This project focuses on natural language analytics, auto-generated visuals, and user-friendly data exploration.

## Objectives
- Enable anyone to explore data instantly using natural language.
- Generate charts automatically based on user questions.
- Build a clean, simple, interactive analytical experience.
- Demonstrate AI-assisted analytics in Power BI.

## Key Features
1. Natural Language Q&A
- Type a question → Power BI generates the correct visual.
- Supports tables, bar charts, line charts, and more.

2. Interactive Visuals
- Two auto-generated Q&A charts.
- One line chart for trend analysis.
- Fully connected to the dataset.

3. Clean, Professional Layout
- Easy for business users.
- Fast insights without formulas.
  
## How It Works
- Load dataset into Power BI.
- Add a Q&A visual to the report.
- Ask a question (e.g. “Total sales by region”).
- Power BI understands the question and builds the visualization automatically.
